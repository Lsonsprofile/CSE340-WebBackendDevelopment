# CSE340-WebBackendDevelopment
Project is centered on Server-Side Fundamentals and web development 
My setup checklist includes Node.js, npm, Git, and my editor (VS Code or WebStorm). I also need Microsoft Teams for group communication. I’ll make sure I can open a terminal inside my editor and know exactly where my project folder lives. My .gitignore must include node_modules/ and .env so I don’t accidentally push sensitive or unnecessary files to GitHub.

On the coding side, I’ll focus on JavaScript basics, file paths, and the request/response cycle. Express routing is key: each route listens for a request, prepares data, and sends a response. EJS templates will help me generate HTML on the server, and partials will keep my code clean by avoiding duplication of headers and footers.

Deployment is another milestone. I’ll push my latest code to GitHub, set the build command to npm install, and the start command to npm start. Render will use these settings to host my project. I’ll confirm that my deployed site looks the same as my local version, with working navigation, CSS, images, and accessibility checks.
